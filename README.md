# Introduction-to-AI
Use the same prompt, get introductions from LLMs. 

## Intent
Would like to use this as a go to resource for someone trying to introduce specific LLMs to a wider audience.

# Prompt
```
I want to design a hands on workshop for how to use you (TOOL). This will be to a group of up to 30 members of a professional team, with some technical proficiency, but ranging in AI Literacy from never used all the way through a tech enthusiast. I would like for this to be as fun and engaging as possible. I would like to have an introductory portion introducing you as an LLM, a high level overview of your capabilities, plus items that have been reported as uniquely yours. Following that, some demo prompts that I can demonstrate some of those capabilities, then a workshop for participants to try. It would be nice for the workshop portion to perhaps be a game using your skills and some rudimentary prompts.

This is to achieve 3 objectives:
- Up the team's familiarity with (TOOL),
- Raise individual and team AI Literacy
- Increase comfort with using AI and LLM's as daily tools in their lives.
```

## Follow up Prompt
```
This is well done, thank you. Could you convert this to md Markup please?
```
